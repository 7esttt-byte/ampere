# ⚡ Ampere v6 - نظام إدارة المولدات الكهربائية

تطبيق ويب احترافي (PWA) لإدارة المولدات الكهربائية الأهلية، مع نظام تسجيل دخول كامل.

## 🌟 المميزات

- 🔐 **نظام تسجيل دخول** كامل مع 100 مستخدم + admin
- 🎨 **أيقونات زجاجية ثلاثية الأبعاد** بتصميم Glass Morphism
- 📱 **PWA** قابل للتثبيت على الجوال
- 🔄 **يعمل بدون انترنت** (Service Worker)
- 📊 **شاشة إحصائيات شاملة** مع رسوم بيانية
- 🖨️ **طباعة وصولات PDF** (A4 + حراري 80mm)
- 💬 **إرسال واتساب** مباشر للمشتركين
- ✏️ **أزرار تعديل/حذف/واتساب** سريعة تحت كل مشترك
- 🗄️ **Sheet ديناميكي** لكل مشترك
- 🔔 **إشعارات Telegram** للمالك

## 🚀 كيفية الاستخدام

1. افتح الرابط على هاتفك
2. اضغط "إضافة إلى الشاشة الرئيسية"
3. سجّل دخول بـ `admin` / `Ampere@2026`
4. اذهب لـ ⚙️ الإعدادات وأدخل رابط Webhook
5. ابدأ بإدخال العمليات!
