# ⚡ Ampere - نظام إدارة المولدات الكهربائية

تطبيق ويب احترافي (PWA) لإدارة المولدات الكهربائية الأهلية، مع دعم كامل للأتمتة عبر n8n.

## 🌟 المميزات

- 🎨 **تصميم عربي احترافي** بألوان أخضر/ذهبي
- 📱 **PWA** - يمكن تثبيته كتطبيق على الجوال
- 🔄 **يعمل بدون انترنت** (Service Worker)
- 📊 **شاشة إحصائيات شاملة** مع رسوم بيانية
- 🔔 **يرسل إشعارات تلقائية** عبر Telegram
- 🗄️ **يحفظ البيانات** في Google Sheets عبر n8n
- 💾 **يحفظ محلياً** كنسخة احتياطية

## 🚀 كيفية الاستخدام

1. افتح الرابط على هاتفك
2. اضغط "إضافة إلى الشاشة الرئيسية" (Add to Home Screen)
3. افتح التطبيق من الأيقونة الجديدة
4. اذهب لـ ⚙️ الإعدادات وأدخل رابط Webhook الخاص بك
5. ابدأ بإدخال العمليات!

## 🛠️ المكونات التقنية

- **Frontend**: HTML5 + CSS3 + Vanilla JavaScript
- **Charts**: SVG للرسوم البيانية
- **Storage**: localStorage (محلي) + n8n + Google Sheets (سحابي)
- **Backend**: n8n Workflow Automation
- **Database**: Google Sheets
